# UI 改动对比

## 改动 1: 导航栏顺序

### 之前
```
┌─────────────┐
│  🤖 Logo   │  → 跳转到 /chat/models
├─────────────┤
│  💬 大模型  │  (第一位)
│  ✨ 智能体  │  (第二位)
│             │
│     ...     │
│             │
│  ⚙️  设置   │
│  🌓 主题    │
└─────────────┘
```

### 之后
```
┌─────────────┐
│  🤖 Logo   │  → 跳转到 /chat/agents ✅
├─────────────┤
│  ✨ 智能体  │  (第一位) ✅
│  💬 大模型  │  (第二位)
│             │
│     ...     │
│             │
│  ⚙️  设置   │
│  🌓 主题    │
└─────────────┘
```

## 改动 2: 聊天记录展示

### 之前（分离）
```
/chat/models                    /chat/agents
┌──────────────────┐           ┌──────────────────┐
│ 💬 大模型对话    │           │ ✨ 智能体对话    │
│ [新对话]         │           │ [浏览智能体]     │
│ [搜索...]        │           │ [新对话]         │
│                  │           │ [搜索...]        │
│ 对话 1           │           │                  │
│ 对话 2           │           │ 对话 A           │
│ 对话 3           │           │ 对话 B           │
└──────────────────┘           └──────────────────┘
    只显示大模型对话              只显示智能体对话
```

### 之后（统一）
```
/chat/models 或 /chat/agents
┌───────────────────────────┐
│ 💬/✨ 大模型/智能体对话   │
│ [浏览智能体] (仅 agent)   │
│ [新对话]                  │
│ [搜索...]                 │
│                           │
│ [全部] [智能体] [大模型]  │  ← 标签切换 ✅
│                           │
│ ✨ 对话 A (智能体)        │
│ 💬 对话 1 (大模型)        │
│ ✨ 对话 B (智能体)        │
│ 💬 对话 2 (大模型)        │
│ 💬 对话 3 (大模型)        │
└───────────────────────────┘
   统一显示所有对话 ✅
```

## 改动 3: 图标尺寸优化

### 对比表格

| 位置 | 之前 | 之后 | 变化 |
|------|------|------|------|
| 导航栏 Logo 容器 | 32px × 32px | 28px × 28px | -12.5% |
| 导航栏 Logo 图标 | 16px × 16px | 14px × 14px | -12.5% |
| 导航栏模块按钮 | 36px × 36px | 32px × 32px | -11% |
| 导航栏模块图标 | 18px × 18px | 16px × 16px | -11% |
| 侧边栏标题图标 | 16px × 16px | 14px × 14px | -12.5% |
| 侧边栏按钮高度 | 36px | 32px | -11% |
| 侧边栏标签高度 | 32px | 28px | -12.5% |
| Agent 头像 (header) | 32px × 32px | 24px × 24px | -25% |

### 视觉效果
- ✅ 更精致、更现代的外观
- ✅ 减少视觉噪音
- ✅ 提升信息密度（能显示更多内容）
- ✅ 保持良好的可点击性和可读性

## 改动 4: 默认路由

### 之前
- 访问 `/` → 自动跳转到 `/chat/models`

### 之后
- 访问 `/` → 自动跳转到 `/chat/agents` ✅

## 新增功能

### 统一侧边栏的标签功能
- **全部**：显示所有对话（大模型 + 智能体）
- **智能体**：仅显示智能体对话
- **大模型**：仅显示大模型对话

### 搜索增强
- 现在可以同时搜索大模型和智能体的对话
- 搜索结果会自动显示对话类型（通过图标区分）

## 用户体验提升

1. **更直观的导航**：智能体作为主打功能放在最前面
2. **统一的历史记录**：不用在两个页面间切换就能看到所有对话
3. **更灵活的筛选**：通过标签快速筛选想要的对话类型
4. **更精致的界面**：减小的图标让整体界面更加优雅
5. **更好的空间利用**：优化后的尺寸让更多信息能够在同一屏幕展示
