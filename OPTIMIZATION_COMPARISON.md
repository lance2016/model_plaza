# LLM Plaza 优化对比表

## 优化前 vs 优化后

| 维度 | 优化前 | 优化后 | 改进 |
|------|--------|--------|------|
| **信息架构** |
| 左侧栏 | 固定宽度,始终显示 | 可折叠,节省空间 | ✅ 50%空间释放 |
| 阅读宽度 | 固定max-w-3xl | 3档可选(窄/中/宽) | ✅ 灵活适配场景 |
| 思考过程 | 默认展开,占据首屏 | 默认折叠,弱化样式 | ✅ 突出最终结果 |
| **视觉层级** |
| 用户/助手区分 | 小头像,相似样式 | 气泡vs卡片,左右对齐 | ✅ 3倍区分度 |
| 标题层级 | 无区分 | H1/H2下划线,明确层级 | ✅ 清晰结构 |
| 段落间距 | 紧凑 | 1em/1.5em统一间距 | ✅ 舒适阅读 |
| 对比度 | muted-foreground 45%/52% | 38%/56% | ✅ WCAG AA级 |
| **交互效率** |
| 消息操作 | 无 | 复制/重新生成 | ✅ 新增功能 |
| 配置可见性 | 隐藏在设置中 | hover查看配置摘要 | ✅ 一目了然 |
| 会话查找 | 滚动查找 | 实时搜索过滤 | ✅ 10倍效率 |
| 输入工具 | 仅发送按钮 | 工具栏(预留扩展) | ✅ 未来可扩展 |
| **可访问性** |
| 间距体系 | 不统一 | 8px spacing system | ✅ 一致性 |
| 字号 | 14px固定 | 14.5px正文,层级化 | ✅ 可读性 |
| 键盘提示 | placeholder内 | 工具栏明确提示 | ✅ 可发现性 |
